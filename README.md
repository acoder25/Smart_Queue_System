## Getting Started

1. Clone the repo:
   git clone https://github.com/your-username/your-repo.git
   cd your-repo

2. Create a root .env (see Environment Variables below), then install and run:
   - Backend:
     cd server
     npm install
     npm run dev
   - Frontend:
     cd client
     npm install
     npm start

## Environment Variables
Create a `.env` file in `server/` (do NOT commit this file). Example:
- PORT=5000
- MONGO_URI=your_mongo_connection_string
- JWT_SECRET=long_random_secret
